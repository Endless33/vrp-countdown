VRP

VRP (Veil Routing Protocol) is a continuity-first networking architecture focused on:

- Session continuity
- Recovery correctness
- Replay containment
- Authority preservation
- Transport migration
- Execution integrity

Core principle:

Session identity is canonical.

Transport is replaceable.
